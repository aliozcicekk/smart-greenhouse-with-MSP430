# smart-greenhouse-with-MSP430
I built a system using the MSP430 that automatically keeps air temperature, soil moisture, and ambient light levels within ideal parameters.

Materials Used:
-Relay
-HC-05 Bluetooth module
-SSD1306 OLED display
-LDR (Light Dependent Resistor) sensor
-Water pump
-DHT11 temperature and humidity sensor
-PWM LED
-Soil moisture sensor
